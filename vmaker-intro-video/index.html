欢迎来到 vMaker。它不是后台系统，也不是静态作品集。
它把 GitHub 公开项目整理成一个可搜索、按语言浏览、持续更新的入口。
数据来自 GitHub API：仓库、语言、主题、星标、分支，以及最近提交。
本地只保留展示规则；事实来源仍然是 GitHub。
首页上半部分是品牌 hero，下半部分是项目索引。
项目按语言分组，搜索覆盖名称、描述、语言和 topic。
左侧时间线读取最新 commit，让入口像一张实时项目地图。
不复制数据库，不增加后台层，只把真实仓库变得更容易进入和理解。
